# Checkbox Group

### CheckboxGroups are form components that allow binary and multiple selections.
---
<br />

## When to use:
- When you need a form component for binary selections with multiple options.

<br />

## When not to use:
- When the effect of the interaction with Checkbox is required to be immediate. Checkboxes should always be used in conjunction with a submit button. To work around cases like this, use Switches.
