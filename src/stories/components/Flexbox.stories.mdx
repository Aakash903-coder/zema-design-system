# FlexBox

### Flexbox is a component that uses the CSS Flexbox API to provide control over the alignment and distribution of elements in a single dimension (row or column).
---
<br />

## When to use:
- To align elements in a single dimension, whether in a row or column.
- When it is necessary to distribute the space between items in a responsive way, adapting to the screen size.
- To center elements horizontally or vertically within a container.
- When it is necessary to control the growth, shrinkage and spacing of child elements in relation to the parent container.

<br />

## When not to use:
- When the layout requires organization in two dimensions (rows and columns simultaneously). For these cases, use `<rds-grid>`.
- For scenarios involving complex layout structures, such as Dashboards or Image Galleries, where two-dimensional control is required.
- In situations where fixed or absolute positioning is necessary to control the layout of the content.
