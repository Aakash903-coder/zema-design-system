# LoadingIndicator

### The LoadingIndicator is a component used to indicate page loading progress, providing a more fluid and informative user experience.
---
<br />

## When to use:
- When navigating between pages or tabs within an application, where the user needs to clearly see the loading progress to know that the transition is in progress;
- In applications where content loading may take time, such as network operations or heavy data loading.

<br />

## When not to use:
- For loading secondary data or actions that do not directly affect the user's main navigation;
- When the user performs interactions that can trigger new loads, such as activating a toggle switch. In these cases, consider using alternatives like **Skeleton** to show content being loaded or **Spinner** to indicate that an action is in progress.
