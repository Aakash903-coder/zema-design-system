# Chip

### Chips help people enter information, make selections, filter content or trigger actions.
---
<br />

## When to use:
- It is necessary to select and/or filter content
- The content that the chip represents can be set or removed by the user.



<br />

## When not to use:
- The content to be shown is *readonly* (in this case, it is best to use the Badge component).
